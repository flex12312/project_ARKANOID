Делаем игру арканоид 
с 2 платформами (бот и игрок) и мячем,
есть 3 уровня сложности для бота , которые меняют его поведение.Сами плафтормы двигаются постоянно. Управление платформой на кнопку S(меняет направление движения). Игра длится до 3 очков , есть перезапуск после конца игры. Также имеетя пасхалка про Стасика. Может быть будет возможность игры друг против друга(но это не точно)
